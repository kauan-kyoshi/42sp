Módulo 3

Automação

Neste módulo, exploraremos como automatizar o uso da inteligência artificial. Você aprenderá a automatizar tarefas repetitivas, categorizar conteúdo automaticamente e criar conexões entre diferentes plataformas.

Para este módulo, você irá precisar de uma chave de API do Gemini. Se você ainda não tem uma, siga estes passos:
1. Acesse o Google AI Studio
2. Crie uma conta ou faça login
3. Gere uma chave de API

O Gemini tem um limite de uso gratuito bastante generoso por minuto.
Você não pagará por utilizá-lo, mas ao conectá-lo com as ferramentas que utilizaremos neste módulo, use a versão do Gemini 2.0 Flash para garantir o maior limite de uso.

Exercício 1 – Notícias do Dia

Para este exercício, utilizaremos a plataforma make.com

O Make é uma plataforma de automação que permite conectar diferentes aplicativos e serviços web de maneira visual e flexível, sem necessidade de código. Através de cenários, você pode criar fluxos de trabalho altamente personalizáveis que executam ações automaticamente quando certos eventos acontecem.

Alguns nomes que é importante saber:
Cenário: a automação completa
Módulo: cada passo / ferramenta da automação
Gatilho: a ação que dá início à automação, atrelada ao primeiro módulo (pode ser o recebimento de um e-mail, um intervalo de tempo, uma nova linha no Google Sheets/Excel)

Atividade

Pasta para entrega: /modulo3/ex1

Arquivo para entrega: diario_<data-e-hora>

Como parte de uma iniciativa para manter toda a equipe atualizada sobre as últimas tendências do mercado, o Diretor de Marketing, conhecido por sua paixão por tecnologia e eficiência, propôs um desafio empolgante. Ele quer implementar um sistema automatizado que forneça um resumo diário das notícias mais relevantes do dia, permitindo que a equipe esteja sempre um passo à frente da concorrência. Sua missão, é criar essa automação utilizando as mais recentes ferramentas de IA e integração de sistemas.

Crie uma automação para acessar diariamente o seu site de notícias preferido, utilizando a IA para criar um resumo das principais notícias, e salvá-lo em um arquivo no seu Google Drive.

Abaixo, um exemplo de como a sua automação pode se parecer:

Neste caso, os seguintes módulos:
1. Requisição HTTP: Tem um gatilho diário para fazer uma busca no site de um jornal.
2. Processador de Texto: Transforma o código da página da internet (html) em um texto comum
3. Google Gemini: Transforma o texto recebido em um resumo de notícias através de um prompt bem redigido
4. Google Docs: Salva o resumo de notícias produzido pelo Gemini em um documento no seu Google Drive.

Bônus: que tal expandir a automação e, ao invés de resumir apenas um site de notícias, a IA agregasse as principais manchetes de dois ou três jornais diferentes? Neste caso, precisaríamos de múltiplas requisições HTTP e múltiplos processadores de texto.

Exercício 2 – Zapier

Para este exercício, utilizaremos a plataforma zapier

Assim como o Make, o Zapier é uma plataforma de automação que permite conectar diferentes aplicativos e serviços.

Alguns nomes que é importante saber:
Zap: a automação completa
Step: cada passo / ferramenta da automação
Gatilho: a ação que dá início à automação, atrelada ao primeiro módulo da automação (pode ser o recebimento de um e-mail, um intervalo de tempo, uma nova linha no Google Sheets/Excel)
Ação: o que cada ferramenta da automação irá executar em seu devido passo.

Neste exercício, faremos uma atividade que conecta o Slack ao WordPress

O WordPress é o nosso website https://orbitmotordrones.42sp.org.br

Você não deve acessar o WordPress diretamente. Use o Zapier para
conectar ao WordPress com os seguintes dados:

Nome de usuário: author
Senha: OrbitMotordrones!!!

Configure a temperatura do prompt do Gemini para 0.5 (zero ponto cinco) para obter resultados mais precisos. A temperatura padrão (1.) pode retornar erros específicos do Zapier

Exercício 3 – Categorização Inteligente de Emails (Bônus)

Para este exercício, utilizaremos a plataforma zapier

Os e-mails da Orbit estão desorganizados. Quando foram criados, a empresa ainda era pequena e o estagiário achou que era uma boa ideia numerá-los, ao invés de dar um nome descritivo. Agora, a Orbit tem milhares de e-mails numerados e ninguém sabe o que eles contêm.

Neste exercício, criaremos uma automação que monitora uma caixa de email, usa a IA para categorizar as mensagens e as encaminha para canais específicos no Slack baseado em seu conteúdo.

Utilize o e-mail designado ao seu grupo para este exercício: Ex: Grupo 1 – orbitmotordrones1@42sp.org.br

Atividade
  1. Monitore a caixa de e-mail designada ao seu grupo.
  2. Use o Gemini para analisar e categorizar o conteúdo em:
    • Suporte (use o canal #orbit-suporte)
    • Vendas (use o canal #orbit-vendas)
    • Elogios (use o canal #orbit-elogios)
  3. Encaminhe cada mensagem para o canal apropriado do Slack. Coloque o número do seu grupo no início da mensagem no slack para identificação.

Exemplo de prompt para categorização com o Gemini:

Você é um classificador de emails. Analise o conteúdo abaixo e categorize em uma das seguintes opções:
SUPORTE (para questões técnicas, problemas ou ajuda)
VENDAS (para oportunidades comerciais ou negociações)
ELOGIOS (para feedback positivo e agradecimentos)

Retorne APENAS a categoria, sem explicações adicionais.
